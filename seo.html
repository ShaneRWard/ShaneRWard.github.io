Search Engine Optimization

The following are guide lines and tips that will help to, as the title suggests, optimize Google's search engine results for your site. It uses several key words throughout your site. Read through these to learn how to increase traffic through your site.

Lookup Keyword Research

  • Use 2-3 keywords for every post. Have multiple similar keywords used througout the page.

    • Keywords

  • Use Yoast plug-in (free) if using Word Press
    1. Make title: Get first key word as close to the front as possible. Try to get second and third in as well in a readable way.
    2. Description:
    3. URL: Use top key word if possible
  • On a page, try to keep only one h1 tag. If title creates it for you, don't add any others, but make sure there is one if it does not.

    • Structure

  • h1 title at top
  • Text: Contains keywords
  • Pinterest image 700x400px. Pings search engines and helps get indexed faster. File name of pinterist picture should be keyword. Alt text also has main keyword. Other images should have same format using second best keyword for second picture, third keyword for third picture, ets.
  • h2 heading: section that supports major idea. Uses similar keywords.
  • Text: Contains keywords
  • h3 heading: section that supports subsections. Uses similar keywords.
  • Text: Contains keywords
  • Video: If you have your own video made on the topic, embed it. If not, use someone elses that is very relevant to the topic. Find what videos Google think are most relevant to it. Make sure it's a good video without too many ads, distractions, CTAs, etc.
  • Text: Contains keywords
  • Look up Miles Beckler's page for structure.
  • Links: Connect links of other work you have that relates and use keywords for those.

    • Google's Web Spiders

  • Bots that Google has made.
  • Crawls the content of every page of your website.
  • Revisits every day.
  • Follows every link and crawls the content there and follows subsequent links.

    • Backlinking

  • Generally not important down the road, but may help early on.
  • Caution: Leaving many links back to page may hurt if overdone. Do not spam.
  • Don't spend a whole lot of effort supporting someone else's content.

    • YouTube

  • Keywords should be in the file name of video when you upload.
  • Keywords in title.
  • Use keywords in intro of video.
  • Use keywords in beginning of the description.
  • Use keywords at end of the description.
  • Avoid linking out from YouTube video unless content is directly connected and vital.
