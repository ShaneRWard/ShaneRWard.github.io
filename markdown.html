Markdown

Markdown is a very simple markup language that is designed to be easily converted into other formats. Mark down files end with the file extension .md or .markdown. It has similarities to that of html. Below are several of the basic elements within markdown.

Syntax Effect Description
# Heading1

Heading1

 Largest heading size. HTML Equivalent to <h1>.
## Heading2

Heading2

 Second largest heading size. HTML Equivalent to <h2>.
### Heading3

Heading3

 Third largest heading size. HTML Equivalent to <h3>.
#### Heading4

Heading4

 Fourth largest heading size. HTML Equivalent to <h4>.
##### Heading5
Heading5
 Fifth largest heading size. HTML Equivalent to <h5>.
###### Heading6
Heading6
 Smallest heading size. HTML Equivalent to <h6>.
  • **bold text**
  • __bold text__
 bold text Styles the enclosed text as bold.
  • *italic text*
  • _italic text_
 italic text Italicizes the enclosed text.
~~scratched out~~ scratched out Puts a line through text to scratch it out.

  • second item
