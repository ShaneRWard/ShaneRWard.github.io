CSS

If a web site was a house, HTML would be its frame, the boards that make up the floor, walls, and ceiling. It gives it shape, but there's not a whole lot to look at otherwise. Cascading Style Sheets, or CSS is what gives it texture, color, and... well, style. CSS is essential to any modern website. Otherwise, it'll look much like a pile of sticks.

CSS files are linked to HTML documents in the head area. From this link, the website refrences the CSS file and gets directions on how to style different HTML elements.

A CSS page should consist of numerous rule sets. Each rule set starts with a selector which is a text label used to identify an element in the HTML document, such as the body, h1-h6 tags, divs, spans, etc. It is followed by a the declaration block whick is enclosed by braces, "{}". Each line within the declaration block is a declaration. Each declaration consists of a property and a value.

CSS can style elements labeled with classes or ids as in the image above. In the HTML document, classes label particular groups of elements, and ids label individual elements for styling. In the CSS style sheet, the selector is preceeded by a "." to indicate a class. The selector preceeded by a "#" is an id.

Though individual pieces of content such as images or text are often styled, the area surrounding them is also styled. The area surrounding this content is broken up into different shells. The area just outside of the content is the padding. Outside of the padding is te border. Finally, the area surounding the border is the margin. This is known as the box model.
