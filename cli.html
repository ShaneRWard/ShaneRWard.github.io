Command Line Interface

Command Line Interface is something every computer user should know at least a little. There are different types of CLI, from Linux, to DOS, to the Windows Command Prompt, but many of the commands are the same or similar. Some CLIs have integrated two or more of these together so either can be used with the same CLI. It's important to know these commands if you're going to be using Git. Git has it's own set of commands that work alongside CLI commands. In the table below are a list of basic commands that you will need to know and a description of their respective properties.

Command Description
dir Lists all files and folders within the current directory.
ls Like dir, lists all files and folders within the current directory, this command is more associated with Linux.
mkdir Creates a new folder or directory.
rmdir Deletes a folder or directory.
rm filename Renames a file.
clear Clears everything from the the CLI window and starts fresh command line.
cd path Changes directory to the listed path or folder.
cd \ Changes directory to the root folder.
cd .. Changes directory one level up from the current folder.
pwd Print, or display, current working directory.
exit Closes CLI terminal
