Hypertext Markup Language

HTML stands for Hypertext Markup Language and is the basic code that makes up web pages. It's generally refered to as the structure of a web page and has less of an effect on the visual outcome. That part is left to CSS and Javascript. Every element in html uses tags to identify the properties of that element. These tags usually come in pairs; an opening tag and a closing tag. Opening tags start with a < symbol, the identifying text such as div or span, then a > symbol. Closing tags are the same except it starts with </ to signify it's closing. As an example a heading tag would look like this:

<h1>Heading Text</h1>

There are several unique elements that do not use closing tags as they are not required for the purpose they serve. We will discuss those elements further down.
